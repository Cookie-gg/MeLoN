## Description

Backend of Cookie_gg's portfolio, [LeMoN](https://github.com/Cookie-gg/LeMoN)
* management of page data (GraphQL + MongoDB Atlas)
* authorization (JWT + Passport)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
